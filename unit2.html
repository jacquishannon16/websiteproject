Internet copyright laws give the original authors or artists the right to exclude others from copying their work or claiming it as their own. While online copyright protection does not protect facts, ideas, systems, or methods of operation, it may protect the way these things are expressed.

Web pages and their contents are covered by copyrights. These rights extend to the:

  • Overall design
  • Links
  • Original text
  • Graphics
  • Audio
  • Video
  • If a web page includes visual content (graphic images such as photos), a person who is blind can not see them. These individuals might access the computer's output text-to-speech technology known as a "screen reader". Graphics must be accompanied (in the html) with alternate text that describes the content of the graphics for anyone who is unable to see visual content. You will see I have added alternative text to my home page image with my name alt="Photograph of Jacqui Shannon".
  • If a web page has buttons, controls, menus, form fields, etc., that require a user to click on them with a mouse, users with physical disabilities who are unable to operate a mouse will be unable to access these controls. All controls should be operable with a keyboard as well. I have addded labels on my form which again assist with usabliity.
  • CSS files - Cascading StyleSheet files can help define font, size, color, spacing, border and location of HTML information on a web page, and can also be used to create a continuous look throughout multiple pages of a website.
  • HTML files - It is used to format text, tables, images, and other content that is displayed on a webpage. HTML files are widely used on the web as most pages within static websites have an ".html" extension.
  • JavaScript files -

    JavaScript is used in Web development to do such things as:

    • change a formatted date
    • Cause a linked-to page to pop up in a new window
    • Cause text or an image to change during a mouse rollover
    • JS file is used mainly to run client side JavaScript code on a webpage. The JS file contains all the HTML HEAD and BODY objects in the tags of an HTML page.
  • GIF - Graphics Interchange Format - the most common image format on the Internet - our browser can display them.
  • JPEG - Another very common image file format, mainly used for photos. File sizes for JPG tend to be larger than a GIF.
