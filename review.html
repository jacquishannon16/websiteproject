Visual Studio Code: is my favorite code editor as I have in the past used Adobe Dreamweaver, Notepad++ and Notepad. The syntax highlighter, makes it very easy to code especially when learning a new programming language and gives intellisense which will complete the code for you, when you start typing. Visually it is also very helpful for a beginner, I really like the color coding and automatic error checking. Debugging directly from the editor which again helps when learning. It has an Interactive Playground it is a special page that goes through common keyboard shortcuts i.e. and concepts like multiple cursors, split panes, and keyboard shortcuts.

Codepen: Is an online tool so you do not need to download it to use it. You can easy code in online and copy paste the code to a local editor for further editing. This tool is good for starting out if you have not used an editor before and very simplistic in its approach. It will also do syntax however you are required to Cntrl space for each piece of code whereaas with Visual Studio Code you only need Sturn it on once. Codepen you can write tiny little bits of HTML/CSS/JS code and get an immediate visual display.
